\# Fitness Tracker



Eine React-App zum Tracken deines Trainingsstatus.  



\## Features

\- Übungen anlegen mit Name + Beschreibung  

\- Trainingshistorie erfassen (Datum, Sätze, Wiederholungen, Gewicht)  

\- Historie in einer Tabelle (neu → alt)  

\- Lokale Speicherung (localStorage)  

\- Modernes, helles Design  

\- Logo als iOS Homescreen-Icon
